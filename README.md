第一次将做的学习项目推送到github上

项目如何运行:
进入buid文件夹
删除所有文件、重新编译
rm -rf *
cmake ..
make

如然后进入bin文件夹
有客户端和服务器两个可执行程序
可在不同终端运行多个服务器
nginx配置文件中配置服务器为6000和6002端口
服务器需在这两个端口上运行

然后在其他终端运行客户端
按照客户端提示操作即可

(项目需安装muduo库、nginx、redis、mysql)
mudo库解耦网络模块和业务模块
nginx解决服务器负载均衡
redis解决用户的跨服务器通信
mysql存储用户数据
